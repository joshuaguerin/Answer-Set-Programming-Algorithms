
# Problem : 2D Convex Hull

## Description

The generator takes in 3 arguments:  number of points ( n ), restriction to first quadrant ( p ), and a maximum distance from the origin ( d ). The number of points is how many nodes will appear as part of the graph. The restriction to first quadrant controls where the points will be placed randomly. The maximum distance determines the maximum distance that a point can be from the origin (Manhattan distance).


## Usage

```python3
python3 ./generate.py -n 11 -p 0 -d 15
```
